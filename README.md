# ANIMICRO: Implementasi Animated Microlearning untuk Peningkatan Efektivitas Pembelajaran Online Berbasis Moodle

<img width="1920" height="1080" alt="Red Illustrative Benefits of Digitalization Presentation" src="https://github.com/user-attachments/assets/b7cdb93e-88f2-4de0-ac5a-12a7a0d84cde" />

ANIMICRO adalah sistem pembelajaran online berbasis Moodle yang mengintegrasikan konsep animated microlearning, yaitu penyampaian materi pembelajaran singkat melalui media animasi interaktif.

Produk ini dirancang untuk membantu peserta didik memahami materi secara cepat, menarik, dan efektif melalui video animasi pendek, kuis interaktif, serta sistem pembelajaran modular yang dapat diakses kapan saja dan di mana saja.

ANIMICRO mendukung proses belajar yang lebih fleksibel dengan tampilan modern dan fitur monitoring pembelajaran secara real-time bagi pengajar.

Fitur Utama
a. Animated Microlearning

Materi pembelajaran disajikan dalam bentuk video animasi singkat berdurasi 3–7 menit untuk meningkatkan fokus dan pemahaman.

b. Modul Pembelajaran Interaktif

Materi dibagi menjadi beberapa topik kecil agar mudah dipelajari secara bertahap.

c. Kuis Interaktif

Setiap modul dilengkapi evaluasi singkat untuk mengukur pemahaman peserta.

d. Learning Progress Tracking

Pengguna dapat melihat progres pembelajaran secara real-time.

e. Responsive Design

Sistem dapat diakses melalui laptop, tablet, maupun smartphone.

f. Gamifikasi

Pemberian badge, poin, dan sertifikat untuk meningkatkan motivasi belajar.

g. Dashboard Pengajar

Memudahkan pengajar dalam mengelola materi, tugas, dan monitoring aktivitas peserta.
